### Dependencias

Paso 1: Instalar las dependencias del proyecto.

```
$ npm install
```

```
npm install --save-dev gulp
```

## Compilar

Paso 3: Compile primero para implementación local web app .

```
npm run start
```


Paso 4: Si va a compilar el backend (port 3000).

```
npm run nodemon
```
